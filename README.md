# 🏡 HomeExchange Bot

Телеграм-бот для обміну житлом між мандрівниками.
Ви їдете до Барселони — хтось із Барселони приїде до вас!

---

## 🚀 Запуск за 2 хвилини

### Варіант 1 — Автоматично (рекомендовано)

```bash
# 1. Розпакуйте архів і перейдіть у папку
cd home_exchange_bot

# 2. Запустіть скрипт — він сам все зробить
bash setup.sh
```

Скрипт сам встановить Docker (якщо немає), запитає токен і запустить бота.

---

### Варіант 2 — Вручну

```bash
# 1. Встановіть залежності
pip install -r requirements.txt

# 2. Скопіюйте і заповніть .env
cp .env.example .env
# Відредагуйте .env — вставте BOT_TOKEN

# 3. Запустіть
python bot.py
```

---

## 🤖 Як отримати токен

1. Відкрийте Telegram → знайдіть **@BotFather**
2. Надішліть `/newbot`
3. Введіть назву: `HomeExchange Bot`
4. Введіть username: `home_exchange_yourname_bot`
5. Скопіюйте токен вигляду `1234567890:AABBcc...`

## 📁 Структура проекту

```
home_exchange_bot/
├── bot.py                  # Точка входу
├── config.py               # Налаштування
├── setup.sh                # Автозапуск
├── Dockerfile
├── docker-compose.yml
├── .env.example
├── database/
│   └── db.py               # SQLite: users, trips, matches, reviews, likes
├── handlers/
│   ├── start.py            # /start, реєстрація, меню
│   ├── trip.py             # Додати поїздку + фільтри типу мандрівника
│   ├── matches.py          # Прийняти/відхилити матч
│   └── reviews.py          # Відгуки, рейтинг, лайки
└── services/
    └── matcher.py          # Алгоритм матчингу
```

## 🔄 Флоу користувача

```
/start
  └── Реєстрація: місто → країна → опис житла

/trip  (або кнопка «Додати поїздку»)
  └── Куди → Країна → Дата виїзду → Дата повернення
       → Кількість гостей
       → Хто ви? [Сім'я / Жінки / Digital nomad / Пара / Соло / Будь-хто]
       → Кого шукаєте? [ті ж категорії]

  🔍 Бот шукає матчі:
     ✅ Географія збігається (A→B і B→A)
     ✅ Дати перетинаються
     ✅ Типи мандрівників сумісні

  📩 Обидва отримують картку матчу з кнопками:
     [✅ Прийняти] [❤️ Зберегти] [❌ Відхилити]

  🎊 Обидва прийняли → бот відкриває контакти

/rating  — ваш рейтинг хоста
/saved   — збережені хости

Після обміну:
  └── Відгук за 4 критеріями (сліпий — відкривається після обох)
      🧹 Чистота
      💬 Комунікація
      📋 Дотримання правил
      ✨ Загальне враження
```

## ⚙️ Docker команди

```bash
# Запустити
docker compose up -d

# Логи в реальному часі
docker compose logs -f

# Зупинити
docker compose down

# Рестарт після змін
docker compose down && docker compose up -d --build

# Резервна копія бази даних
docker run --rm -v home_exchange_bot_bot_data:/data \
  -v $(pwd):/backup alpine tar czf /backup/db_backup.tar.gz /data
```
